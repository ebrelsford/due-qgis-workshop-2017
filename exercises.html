Exercise 1: styling layers

  1. Download the data for the day.
  2. Open QGIS.
  3. Open the shapefile in School Districts.
  4. Open the shapefile in Public School Locations.
  5. Style both layers.
  6. Add labels to the school districts layer.

Exercise 2: using print composer

  1. Open the shapefile in School Districts from Part 1 if you have closed it.
  2. Open the shapefile in Public School Locations from Part 1 if you have closed it.
  3. Add a base map.
  4. Create a print composer, adding the map, a title, and a legend.
  5. Export a PDF of your map.

Exercise 3: opening CSVs, making selections

  1. Open the CSV in the NYPD Motor Vehicle Collisions folder, setting the projection appropriately.
  2. Select only those features where more than one person was injured. This is just to confirm that you can do so—clear the selection once you've done this.
  3. Filter the layer to only those features where more than one person was injured.
  4. Save the filtered features to a shapefile.

Exercise 4: counting points in polygons, making choropleths

  1. Open the shapefile in School Districts.
  2. Open the filtered shapefile of NYPD Motor Vehicle Collisions from the previous exercise.
  3. Count the number of crashes in each school district: Vector > Data Management Tools > Join attributes by location.
  4. Create a choropleth: style the resulting layer using a Graduated style on the count column.

Exercise 5: putting it all together

Make a PDF map using the techniques we talked about today.

  1. Find at least two data sources from the NYC Open Data Portal. I recommend filtering the datasets to only show maps. Look for one dataset that is polygons and one that is points. For example, borough boundaries for points and wifi hotspot locations for points.
  2. Load both datasets in QGIS.
  3. Add a base map.
  4. Filter one or both datasets.
  5. Use make a choropleth of the number of points in each polygon.
  6. Create a print composer, adding the map, a title, and a legend.
  7. Export a PDF of your map.
  8. Put your map in this Google Drive folder.
